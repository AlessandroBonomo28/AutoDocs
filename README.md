# AutoDocs
 Automatic tool for work documentation and reporting
## Quick start
```
python -venv env
```
- activate env linux
```
source env/bin/activate
```
- activate env windows
```
source ./env/Scripts/activate
```
```
pip install -e .
```
## .env
```
VAULT_DIRECTORY=C:\...
OPENAI_API_KEY=your-key
```
